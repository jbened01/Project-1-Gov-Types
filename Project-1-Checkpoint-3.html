API Function

takes a 3-letter country code and returns a GDP data frame for that country.

get_gdp <- function(country, key = my_key) {
  
  url <- glue("https://api.api-ninjas.com/v1/gdp?country={country}&X-Api-Key={key}")
  my_data <- safely(fromJSON)(url)
  
  if (is.null(my_data$result)) {
    return(NULL)
  } else {
    return(my_data$result)
  }
}

Data Summary #1

GDP by Different Regimes

prop <- function(variable){
  meta_data |>
    filter(.data[[variable]] == 1) |>
    summarise(
      avg_gdp = mean(gdp_per_capita_nominal, na.rm = TRUE),
      n = n()
    )
}

vars <- c("is_monarchy", "is_commonwealth", "is_presidential",
          "is_colony", "is_communist")

result <- map_dfr(vars, function(var) {
  prop(var) |>
    mutate(variable = var)
})

The differences in sample sizes (n) across groups suggest that some categories, such as presidential regimes, are much more widely represented while others are based on less observations which may influence the averages.

Life Expectancy by GDP

ggplot(
  meta_data |> drop_na(gdp_per_capita_nominal, Life.expectancy),
  aes(x = gdp_per_capita_nominal, y = Life.expectancy)) +
  geom_point() +

  labs(
    title = "Relationship Between GDP per Capita and Life Expectancy",
    subtitle = "Countries over time",
    x = "GDP per Capita (USD)",
    y = "Life Expectancy (years)"
  )
`geom_smooth()` using formula = 'y ~ x'

The chart shows a clear positive relationship between GDP per capita and life expectancy, with higher income countries experiencing longer lifespans. At lower income levels life expectancy varies more than higher income levels, suggesting other factors also play a role, while middle and high income countries show more consistent and higher life expectancy. The trend line indicates that increases in income are linked to longer lives.
